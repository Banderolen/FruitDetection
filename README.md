Ниже представлена реализация MVP приложения Gradio.

Инструкция:

Приложение представлено в виде 2 окон с изображениями.

Для работы нужно сначала загрузить картинку для детекции повреждений в левое окно. Для этого нужно перетащить файл картинки в окно.

Чтобы начать процесс детекции повреждений, нужно нажать кнопку Submit. После завершения процесса распознования в правом окне появится итоговое изображение с обведенными в рамки повреждениями и их названиями.

Чтобы очистить результат и загрузить новое изображение, нужно нажать кнопку Clear.
